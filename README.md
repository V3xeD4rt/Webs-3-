## Часть 1

### Задание 1. Проверка окружения
Создан простой PHP-скрипт, который выводит строку `Hello, PHP!` с помощью `echo`. Скрипт используется для проверки корректной установки и работы PHP.

### Задание 2. Строгая типизация
В начале файла включена строгая типизация (`declare(strict_types=1)`). Реализована функция `sum(int $a, int $b): int`, выполняющая сложение двух целых чисел. Также проверено поведение функции при передаче аргументов неверного типа.

### Задание 3. Обработка ошибок деления
Реализована функция `divide(float $a, float $b): ?float`, которая возвращает `null` при попытке деления на ноль. В случае ошибки выводится соответствующее сообщение.

### Задание 4. Оператор match
Написана функция `grade(int $score): string`, возвращающая буквенную оценку (`A`, `B`, `C`, `F`) в зависимости от количества баллов с использованием оператора `match`.

### Задание 5. Null-safe оператор
Показан пример использования null-safe оператора `?->` при вызове метода у объекта, значение которого равно `null`, и объяснено отличие от обычного вызова метода.

### Задание 6. Анонимные функции и стрелочные функции
С помощью `array_map` и стрелочной функции создан массив квадратов чисел. Также реализована функция, определяющая, является ли число чётным или нечётным.

### Задание 7. Мини-задача «FizzBuzz»
Реализован скрипт `quiz.php`, принимающий число через `$_GET`. В зависимости от значения выводится `Fizz`, `Buzz`, `FizzBuzz` или само число.

## Часть 2

### Задание 8. Класс Product
Создан класс `Product` с публичными свойствами `name` и `price`. Создан объект класса и выведены значения его свойств.

### Задание 9. Класс User
Создан класс `User` с полями `name` и `email`. Добавлен метод `getInfo()`, который возвращает строку с информацией о пользователе. Метод вызывается для созданного объекта.

### Задание 10. Интерфейсы и реализация
Определён интерфейс `Payable` с методом `pay()`. Реализованы классы `CashPayment` и `CryptoPayment`, которые по-разному обрабатывают оплату. Оба класса вызываются через общий интерфейс.

### Задание 11. Статические методы
Создан класс `MathUtils` со статическим методом `square()`, который вычисляет квадрат переданного числа. Метод вызывается без создания объекта класса.

### Задание 12. Класс StringUtils
Реализован класс `StringUtils` со статическим методом `isPalindrome()`, который проверяет, является ли строка палиндромом. Метод протестирован на нескольких строках.
